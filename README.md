# InstagramAutoMeme
Downloads and posts a meme to insta from a subreddit of your choice, in any timeframe you want.

# Installation

- Download and unzip the project
- run `pip install -r requirements.txt`
- Put your login information into `secrets.py`
- run `__main__.py`
